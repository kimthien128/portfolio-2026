Thêm assets cho trang portfolio:

- assets/me.jpg — ảnh portrait (kích thước gợi ý: 512x512, webp/avif hoặc jpg)
- cv.pdf — đặt ở gốc dự án nếu muốn link tải `/cv.pdf`

Hướng dẫn nhanh:
1. Thêm file ảnh: `assets/me.jpg`
2. Thêm file CV: `cv.pdf` (ở root)
3. Mở `index.html` và thay `[Tên của bạn]`, email, các link dự án.
